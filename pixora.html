⚙ Instrucciones del servidor local

Pixora necesita que el servidor Python esté corriendo en tu máquina. Sigue estos pasos:

1. Instala las dependencias:

pip install flask flask-cors opencv-contrib-python numpy pillow

2. Ejecuta el servidor (en la misma carpeta que pixora_server.py):

python pixora_server.py

3. El servidor correrá en http://localhost:5000. Los modelos FSRCNN se descargan automáticamente la primera vez (~1 MB cada uno).

Nota: usa opencv-contrib-python (no opencv-python), ya que contiene el módulo dnn_superres.
