⚙ Instrucciones del servidor local

Esta herramienta requiere el servidor Pixora con ffmpeg instalado en tu sistema.

pip install flask flask-cors pillow

Instala ffmpeg:

# macOS: brew install ffmpeg
# Ubuntu/Debian: sudo apt install ffmpeg
# Windows: descarga desde https://ffmpeg.org/download.html

Ejecuta el servidor:

python pixora_server.py
