⚙ Instrucciones del servidor local

Esta herramienta utiliza el servidor Pixora con la librería rembg. Instala las dependencias:

pip install flask flask-cors rembg pillow numpy

Ejecuta:

python pixora_server.py

El servidor corre en http://localhost:5000. El modelo U2Net se descarga automáticamente (~170 MB) la primera vez.
